# CP/M80_Z80-MBC2/Modula-2/ASCIIART
\
H Drive (DS0N07.DSK)に インプリメント済みの TURBO Modula-2 1.0 と入れ替える。（要バックアップ）
\
H> M2 で TURBO Modula-2 IDE を起動
\
Compile で ASCIIART.MOD をコンパイル
\
Run でオブジェクト実行
