# CP/M80_Z80-MBC2/ADA/ASCIIART
\
G Drive (DS0N06.DSK)に インプリメント済みの JANUS / ADA version 1.5.0 と入れ替える。（要バックアップ）
\
G> janus asciiart でコンパイル
\
G> jlink asciiart でオブジェクト生成
\
G> asciiart でオブジェクト実行
