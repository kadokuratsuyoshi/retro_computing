# CP/M86_V20-MBC/C/ASCIIART
\
G Drive (DS1N07.DSK)に インプリメント済みの Aztec C 8086 3.20d と入れ替える。（要バックアップ）
\
G> CC ASCIIART.C でコンパイル
\
G> LN ASCIIART.O M.LIB C.LIB でオブジェクト生成
\
G> asciiart でオブジェクト実行
\
